おすすめレシピ集

基本のパンレシピ

基本の食パン（1斤）

材料
  • 強力粉：280g
  • 水：190ml
  • 砂糖：17g
  • 塩：5g
  • バター：10g
  • ドライイースト：3g
作り方
  1. パンケースに水、砂糖、塩を入れる
  2. 強力粉を加え、中央にくぼみを作る
  3. くぼみにドライイーストを入れる
  4. バターを角に置く
  5. 「食パン」メニューを選択してスタート

ポイント：水温は室温程度に。冬は少し温めに、夏は冷たくするとよい発酵になります。

レーズン食パン（1斤）

材料
  • 強力粉：280g
  • 水：190ml
  • 砂糖：20g
  • 塩：5g
  • バター：15g
  • ドライイースト：3g
  • レーズン：60g
作り方
  1. 基本の食パンと同じ手順で材料を入れる
  2. 具材投入ケースにレーズンを入れる
  3. 「レーズンパン」メニューを選択
  4. 自動でレーズンが投入される

ポイント：レーズンは事前にぬるま湯でもどして水気を切ると、よりふっくらします。

全粒粉パン（1斤）

材料
  • 強力粉：210g
  • 全粒粉：70g
  • 水：200ml
  • 砂糖：12g
  • 塩：5g
  • バター：15g
  • ドライイースト：3g
作り方
  1. 全粒粉と強力粉をよく混ぜる
  2. 水を少し多めに加える
  3. 「全粒粉パン」メニューを選択
  4. 焼き色は「薄め」がおすすめ

ポイント：全粒粉は水分を吸いやすいので、通常より水を10ml多めに入れます。

パン以外の活用レシピ

うどん（4人分）

材料
  • 薄力粉：400g
  • 水：180ml
  • 塩：20g
作り方
  1. 塩を水に溶かす
  2. パンケースに薄力粉、塩水を入れる
  3. 「うどん」メニューでこねる
  4. 取り出して30分寝かせる
  5. のばして切る

ポイント：生地は少し固めに仕上がりますが、コシのある美味しいうどんになります。

お餅（約20個）

材料
  • もち米：3合
  • 水：580ml
作り方
  1. もち米は一晩水に浸す
  2. 水を切ってパンケースに入れる
  3. 分量の水を加える
  4. 「餅」メニューを選択
  5. できあがったら片栗粉をまぶして成形

ポイント：つきたては非常に熱いので、やけどに注意してください。

イチゴジャム（約200g）

よくある失敗と対処法

パンの仕上がりに関する問題

パンが膨らまない

主な原因
  • イーストの期限切れや保存状態不良
  • 水温が高すぎる（35℃以上）
  • 塩とイーストが直接触れている
  • 材料の計量ミス
対処法
  • 新しいイーストに交換する
  • 水温を室温程度（20-25℃）にする
  • イーストと塩を離して入れる
  • デジタルスケールで正確に計量する

パンが固い・パサつく

主な原因
  • 強力粉の種類が適さない
  • 水分量が不足している
  • 焼きすぎ
  • 冷却後の保存方法が悪い
対処法
  • パン用強力粉を使用する
  • 水分を10ml程度増やす
  • 焼き色を「薄め」に設定する
  • 完全に冷めたらビニール袋で保存

パンが凹む

主な原因
  • 水分が多すぎる
  • 発酵オーバー
  • 室温が高すぎる
  • イーストが多すぎる
対処法
  • 水分量を10ml程度減らす
  • 室温25℃以上の場合は水を冷やす
  • イーストを0.5g程度減らす
  • 「ソフトパン」コースを試す

焼きムラがある

主な原因
  • パン羽根の取り付け不良
  • パンケースの汚れ
  • 生地の偏り
  • 機器の設置場所
対処法
  • パン羽根をしっかり取り付ける
  • パンケースを清潔に保つ
  • 材料を均等に入れる
  • 水平で風通しの良い場所に設置

操作・機能に関する問題

動作音が大きい

主な原因と対処法
  • 設置場所：不安定な場所→水平で安定した場所に移動
  • パン羽根：汚れや摩耗→清掃または交換
  • 材料量：入れすぎ→規定量を守る
  • 時間帯：夜間使用→タイマー時間を調整

タイマーが作動しない

チェックポイント
  • 電源プラグが確実に差し込まれているか
  • タイマー設定時刻が現在時刻より前になっていないか
  • 使用中に停電があったか
  • メニューによってはタイマー使用不可

イーストが自動投入されない

確認事項
  • イーストケースにイーストが正しく入っているか
  • イーストケースの蓋がしっかり閉まっているか
  • イーストが固まっていないか
  • メニューが自動投入対応コースか

季節別調整のコツ

冬場（室温15℃以下）

  • 水温を30℃程度に温める
  • 発酵時間を延長する
  • 室温の暖かい場所に設置
  • イーストを0.5g程度増やす

夏場（室温30℃以上）

  • 冷水（10℃程度）を使用
  • イーストを0.5g程度減らす
  • エアコンの効いた場所に設置
  • 発酵オーバーに注意

梅雨時期（高湿度）

  • 水分を5-10ml減らす
  • 除湿器を活用
  • 材料の保存に注意
  • カビ防止の為、使用後は完全乾燥

よくある質問

初心者におすすめのホームベーカリーは？

初心者にはシロカ SHB-122またはパナソニック SD-SB4をおすすめします。どちらも操作が簡単で失敗しにくく、基本的なパン作りには十分な機能を持っています。価格も手頃で、イースト自動投入機能があるため、タイマー機能も安心して使えます。

ホームベーカリーで作れるものは？

基本的な食パンはもちろん、フランスパン、全粒粉パン、レーズンパンなどの各種パン類、さらにピザ生地、うどん、パスタ生地、お餅、ジャム、甘酒など多彩なメニューが作れます。高機能機種では50種類以上のメニューを搭載したものもあります。

1斤と2斤、どちらを選ぶべき？

家族構成に合わせて選ぶのがベストです。1〜2人なら1斤、3〜4人なら1.5斤、5人以上なら2斤が目安です。ただし、シロカSHB-122のように1斤・2斤を切り替え可能な機種なら、状況に応じて使い分けできて便利です。

電気代はどのくらいかかる？

1回の使用（約4時間）で約6〜8円程度です。消費電力は機種により異なりますが、多くのホームベーカリーは500〜600W程度。1ヶ月毎日使っても電気代は200〜250円程度と非常に経済的です。

夜中にセットしても大丈夫？音は気になる？

最近のホームベーカリーは静音設計が進んでおり、特にパナソニックやタイガーの上位機種は非常に静かです。ただし、混ぜている時間（最初の30分程度）は多少音がするので、寝室から離れた場所に設置することをおすすめします。

材料はどこで買える？コストは？

基本材料（強力粉、ドライイースト、砂糖、塩、バター）はスーパーやネット通販で購入できます。1斤あたりの材料費は約65円程度。市販の食パン（平均240円）と比べると、年間2万円以上の節約が可能です。

お手入れは面倒？

基本的なお手入れはパンケースとパン羽根を洗うだけで簡単です。ただし、フッ素コーティングを傷つけないよう柔らかいスポンジで優しく洗い、食器洗浄機は使用不可です。週1回程度の庫内清掃も合わせて行えば長く使えます。

失敗することはある？

正しい材料と分量で作れば失敗はほとんどありません。よくある失敗は「イーストの期限切れ」「水温が高すぎる」「材料の計量ミス」などです。デジタルスケールで正確に計量し、新鮮な材料を使えば、初心者でも美味しいパンが作れます。

どのくらいで元は取れる？

使用頻度にもよりますが、週2〜3回使えば約10ヶ月で元が取れる計算です。例えば15,800円のシロカSHB-122なら、週3回使用（年間156斤）で年間27,300円の節約となり、約7ヶ月で回収できます。

保証期間・アフターサービスは？

多くのメーカーで1年間の保証が付いています。パナソニックや象印などの大手メーカーは全国にサービス拠点があり、修理や部品交換にも対応。特にパン羽根やパンケースなどの消耗品は別売りで購入できるので長期使用が可能です。
